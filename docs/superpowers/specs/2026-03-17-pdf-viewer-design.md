# PDF 教材预览功能设计文档

## 概述

在首页增加 PDF 教材预览功能，自动加载 `data/pdfs` 目录中的 PDF 文件，并提供在线预览能力。

## 目标

- 不干扰现有测验功能
- 使用前端 PDF 组件（PDF.js）渲染，不依赖浏览器原生能力
- 网格卡片式布局展示 PDF 列表
- 支持翻页、缩放、下载等基础预览功能

## 用户流程

```
首页 → 点击"教材预览"卡片 → PDF 列表页 → 点击卡片 → PDF 预览页
```

## 页面设计

### 1. 首页入口 (index.html)

新增"教材预览"卡片，与"错字本"、"打印卡片"并列：

```
┌─────────────────────────────────────────────────────────────┐
│  首页                                                       │
│  ┌─────────────┐ ┌─────────────┐ ┌─────────────────────┐   │
│  │  错字本     │ │  打印卡片   │ │  📚 教材预览 (新)   │   │
│  └─────────────┘ └─────────────┘ └─────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

### 2. PDF 列表页 (pdfs.html)

网格布局，每行 3 列（桌面端），每卡片包含：
- PDF 图标/缩略图
- 文件名（截断显示）
- 文件大小（人类可读格式）
- 点击打开预览

### 3. PDF 预览页 (pdf-viewer.html)

使用 PDF.js 渲染：
- 顶部工具栏：返回按钮、文件名、页码导航、缩放控制、下载按钮
- 中间：PDF 渲染画布
- 底部：页码指示器

## 技术方案

### 后端 API

**GET /api/pdfs**

扫描 `data/pdfs` 目录，返回文件列表：

```json
{
  "pdfs": [
    {
      "filename": "义务教育教科书·语文二年级上册.pdf",
      "size": 14038295,
      "size_human": "13.4 MB",
      "path": "/data/pdfs/xxx.pdf"
    }
  ]
}
```

### 前端实现

**PDF.js 引入**
- CDN: `https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.min.js`
- Worker 配置: `pdfjsLib.GlobalWorkerOptions.workerSrc`

**核心功能**
- 异步加载 PDF 文档
- 渲染指定页面到 Canvas
- 支持翻页（上一页/下一页）
- 支持缩放（放大/缩小/适应宽度）
- 支持下载原始文件

## 路由设计

| 路由 | 说明 |
|------|------|
| `/pdfs` | PDF 列表页 |
| `/pdf-viewer?file=xxx.pdf` | PDF 预览页 |
| `/api/pdfs` | 获取 PDF 文件列表 API |

## 边界情况处理

| 场景 | 处理方式 |
|------|---------|
| 空目录 | 显示"暂无教材文件"提示 |
| 文件读取失败 | 显示错误提示，提供刷新按钮 |
| 文件名过长 | 截断显示，hover 显示完整名 |
| 移动端适配 | 简化工具栏，支持手势缩放 |
| 大文件加载 | 显示加载进度条 |
