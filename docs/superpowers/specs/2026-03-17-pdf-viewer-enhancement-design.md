# PDF 预览器增强设计文档

## 问题描述

当前 PDF 预览功能存在以下问题：
1. 教材预览页面的文件名标题显示不全，被截断
2. 缺少页码缩略图预览，无法快速跳转到指定页面
3. 缺少全屏预览模式
4. 缺少双页面预览模式（类似书本对开页）

## 解决方案

### 1. 标题显示优化（PDF 列表页 pdfs.html）

**修改内容：**
- 卡片文件名区域从单行截断改为两行显示
- 使用 CSS `line-clamp-2` 限制最大两行
- 卡片高度自适应增加

**代码变更：**
```html
<!-- 原代码 -->
<h3 class="text-lg font-semibold text-gray-800 truncate">...</h3>

<!-- 新代码 -->
<h3 class="text-lg font-semibold text-gray-800 line-clamp-2">...</h3>
```

### 2. 侧边栏页码缩略图（PDF 预览页 pdf-viewer.html）

**布局调整：**
```
┌─────────────────────────────────────────────────────────────┐
│  工具栏                                                     │
├──────────────┬──────────────────────────────────────────────┤
│  📄 第 1 页  │                                              │
│  📄 第 2 页  │           PDF 主显示区域                     │
│  📄 第 3 页  │           (单页/双页模式)                    │
│  ...         │                                              │
│  [可滚动]    │                                              │
└──────────────┴──────────────────────────────────────────────┘
```

**功能：**
- 左侧固定宽度侧边栏（200px）
- 显示每页缩略图（宽度 140px）
- 当前页高亮显示（绿色边框）
- 点击缩略图直接跳转到对应页
- 侧边栏可折叠/展开（按钮控制）

**实现方式：**
- 使用 PDF.js 的 `getPage()` 和 `render()` 生成每页缩略图
- 缩略图 scale 设为 0.2 以减小资源占用
- 懒加载：仅渲染可见区域缩略图

### 3. 全屏预览

**实现方式：**
- 使用 Fullscreen API: `document.documentElement.requestFullscreen()`
- 工具栏添加全屏切换按钮
- 快捷键支持：F11 或 F
- 全屏模式下隐藏浏览器 UI

**代码示例：**
```javascript
function toggleFullscreen() {
    if (!document.fullscreenElement) {
        document.documentElement.requestFullscreen();
    } else {
        document.exitFullscreen();
    }
}
```

### 4. 双页面预览

**模式切换：**
- 工具栏添加"单页/双页"切换按钮
- 双页模式同时显示左右两页（对开书本效果）
- 首页和末页特殊处理（单页显示）

**双页布局逻辑：**
```
当前页码为 1（首页）:   显示 [第 1 页]
当前页码为偶数:         显示 [第 N 页] [第 N+1 页]
当前页码为奇数(>1):     显示 [第 N-1 页] [第 N 页]
```

**渲染实现：**
- 使用两个 canvas 元素并排显示
- 双页模式下缩放比例自动调整为 0.7
- 页码导航按钮自动跳转 2 页

### 5. 工具栏按钮调整

**新增按钮：**
- 双页/单页切换按钮
- 全屏按钮
- 侧边栏折叠/展开按钮

**布局：**
```
[返回] [文件名] | [<] [1 / 50] [>] | [-] [100%] [+] | [侧边栏] [单/双页] [全屏] [下载]
```
